Como foi feito o site?

O site é formado basicamente de arquivos HTML simples (XHTML 1.0 Transitional), arquivos CSS e arquivos Javascript. Apenas em algumas seções é usado o PHP para tratamento dos formulários e conteúdo dinâmico da barra lateral.

O que se destaca no site é o uso extenso dos recursos do JQuery e de AJAX

A estrutura

O site é composto da página inicial, que funciona como um esqueleto de todo o site. Conforme o usuário navega no site, somente o conteúdo dinâmico interno é modificado. Isso facilitou na divisão das tarefas de cada membro do grupo, pois o template mestre é o arquivo index.html. Então, cada membro poderia pegar uma parte do site para edição.

Os arquivos de conteúdo contém somente as tags básicas. É com o AJAX que estes arquivos são carregados na navegação.
